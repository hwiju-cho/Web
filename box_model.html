display: inline

share the same line with other elements
It only takes up the width as much as the width of the content.
properties such as width, height, margin-top, and padding cannot be applied.



display: block

Don't share the same line with other elements.
Takes up the entire width of the web page.
When placing multiple block-level elements, they are placed in the vertical direction.
properties such as width, height, margin, and padding can be applied.

display: inline-block

Use properties such as width, height, margin, and padding while placing elements on the same line
